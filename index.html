Summary: Weather and the Equator

The fundamental ask of this assignment was, "What's the weather like as we approach the equator?" For this project I used Python citipy to pull the names of no fewer than 500 different cities. Using those data, I queried the OpenWeatherMap.org API to pull each city's country, latitude, longitude, along with the real-time cloud cover, humidity, maximum temperature, and wind speed. I then used MatPlotLib to create scatter plots for each cloudiness, humidity, maximum temperature, and wind speed compared against the city's latitude. The data in the visualizations on this site were pulled on February 5, 2019, with a total of 560 cities. The complete data for these visualizations can be found here.

For cloudiness and maximum temperature I used Google Maps to create heat maps of the data, which can be found on their respective pages.
